Hyperlinks

Knowing how to create a linked document is important.
  • The <a> <\a> tag stands for "Anchor Link"
  • Needs a hyper-reference AND content
    • href: Reference to location of new content
    • content: The "clickable" part (text or image)

Types of Links

  1. Absolute Links
  2. Relative Links
  3. Graphical Links
  4. Internal Links

Absolute Links

Used to connect to resources in other domains. In the href attribute of the tag the full URL of the resource has to be specified. One example might be your quest to list some top ranked universities. Since you want to connect to universities themselves not files you wrote, you would use an absolute reference/link.

  • Lahore University of Management Sciences (LUMS)
  • National university of Science and Technology (NUST)
  • Pakistan Institute of Engineering & Applied Sciences (PIEAS)

Advantage of Absolute Links: Always linked to up-to-date information on that page.
Disadvantage of Absolute Links: It is up to you to make sure that links do not change.

Relative Links

Used to connect to local resources/files. In the href attribute of the tag only the folder (if required, means if file is not in same folder) and filename is specified (not comlete URL). Some examples of when you might use relative link would be in your navigation bar or links to your images.

  • Flag of Pakistan

Graphical Links / Images as Links

More and more people are moving away from textual prompts to ones that are graphics based. This is because the page looks "nicer". I can redo the relative link above using a picture instead of words.

Internal Links

Used to link to different location (not resource/file) in the same file. In href attribute of the tag the address of location to be linked is specified. (i.e #...)
