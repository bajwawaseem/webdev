Tables

  • Tables are meant for tabular data, not for layout
  • Make sure to sketch your layout before you code
  • Decide on number of rows and columns
  • Decide if any cell my span multiple rows or columns
  • Syntax:
    • <table>   </table>   (The container tag)
    • <tr>   </tr>   (row)
    • <td>   </td>   (column)
  • We can use attribute "border" in start tag of "table tag"
  • To add table headings (headings of columns), we can use "<th>   <\th>" tag
  • We can add captions to tables using "<caption>   <\caption>" tag. No matter in where you place the "caption" tag inside the "table" tag, caption will be above the table.
